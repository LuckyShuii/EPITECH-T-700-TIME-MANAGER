# Design System - TimeTracker

## Couleurs

### Primaire (Violet)
- `primary-500`: #667eea (principal)
- Usage: Boutons principaux, liens, accents

### Secondaire (Violet foncé)
- `secondary-500`: #764ba2
- Usage: Éléments secondaires, hover states

### Gradients
- `bg-gradient-primary`: Violet → Rose
- Usage: Fonds, boutons CTA

## Composants

### Buttons
- Primaire: `btn bg-gradient-primary text-white`
- Secondaire: `btn btn-outline border-primary-500`

### Cards
- Standard: `bg-white rounded-3xl p-12 shadow-card`
- Hover: ajoutez `shadow-card-hover`

### Spacing
- Padding card: `p-12` (3rem)
- Gap éléments: `gap-4` ou `gap-6`

## Typographie

### Titres
- H1: `text-3xl font-extrabold`
- H2: `text-2xl font-bold`

### Corps
- Normal: `text-base text-gray-600`
- Subtile: `text-sm text-gray-500`
